How to start Oracle MoviePlex demo

1. Run /home/oracle/movie/moviedemo/nosqldb/scripts/startDemo.sh
2. After succesfully running above script, open firefox browser 
3. Connect to 'Oracle MoviePlex' demo application by using this URL: 
    http://localhost:7001/bigdatademo-UI-context-root/login.jsp
4. Login using any one of the 100 user accounts:
    username: guest1/guest2/.../guest100
    password: Welcome1

How to Reset Oracle MoviePlex demo

1. Run /home/oracle/movie/moviedemo/nosqldb/scripts/resetDemo.sh
